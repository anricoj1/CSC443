To calculate the statistics, the script will need to collect the frequencies of each submitted answer for each question. The frequencies are to be stored in the Results.txt file (follow the format). The frequencies for the example above are:

For example, 5 in the first row means that there were 5 users that selected the third radio button (denoted 'neutral') for the first question.

To calculate the average for any question this formula can be used:
Avg =  1·f1+2·f2+3·f3+4·f4+5·f5
 
total
where
total =  f1+f2+f3+f4+f5
f1,f2,f3,f4,f5 are the collected frequencies for the specific question.

For our example the average for the first question given the frequencies:
f1,f2,f3,f4,f5 = 0,0,5,2,0

total =  0+0+5+2+0  = 7

Avg =  1·0+2·0+3·5+4·2+5·0  = 3.2857...~=3.29
 
total
